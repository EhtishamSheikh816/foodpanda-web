# 🍽️ Foodpanda Clone Website

Welcome to the **Foodpanda Clone** – a responsive food ordering web app built using **HTML**, **CSS**, and **JavaScript** with full use of **LocalStorage**, **SweetAlert**, and **Google Fonts**.

---

## ✨ Features

- ✅ Beautiful and modern **Foodpanda-inspired UI**
- 🍔 Add restaurant items dynamically
- 📦 Store and retrieve data with **LocalStorage**
- 🎨 Styled with custom **CSS** and **Google Fonts**
- 💬 Interactive alerts using **SweetAlert**
- 📱 Fully **responsive design** (mobile + desktop)
- 📝 Real-time form validation and user feedback

---

## 🚀 Technologies Used

| Tech         | Description                           |
| ------------ | ------------------------------------- |
| HTML5        | Structure of the web pages            |
| CSS3         | Custom design, layout, responsiveness |
| JavaScript   | DOM manipulation, data handling       |
| LocalStorage | Store user input and persist data     |
| SweetAlert   | Beautiful popup alerts                |
| Google Fonts | Clean and modern typography           |
